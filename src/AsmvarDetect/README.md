Output format of 'SVD' program: 
[1]Target Id
[2]Target start
[3]Target end
[4]Target region size
[5]N ratio of target in the region
[6]Target fa length
[7]Query id
[8]Query start
[9]Query end
[10]Query region size
[11]N ratio of query in the region
[12]Query fa length
[13]Mapped strand
[14]Variant type
